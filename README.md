## VisionArm

## VisionArm Overview

The VisionArm is a robotic arm in an articulated configuration with five degrees of freedom, made from servo motors, several parts from my computer and an Xbox 360 controller.

I have developed my interest in this project further through trying to learn more about robotics and machine vision, both of which were relatively new concepts to me before creating this project. Therefore, almost everything I can do with the robotic arm is a result of trial-and-error, through trial and error on this project and gaining further understanding of each through experience.

At this moment I have the ability to control the robotic arm through the Xbox 360 controller being used as the interface from computer to controller to robotic arm. However, it is my hope that I will eventually be able to expand the robotic arm’s functionality by utilizing a machine vision system that will provide the arm with the ability to see objects and then automatically perform tasks with them.

I created the robotic arm because I wanted to create a challenge for myself and find out how robots actually work in reality. Rather than simply watching videos or reading about the subject, I wanted to take the next step to see if I could create something on my own.

Throughout the process of creating the VisionArm, I have learned several different skills directly related to developing and creating robotic arms:

Mechanical design/robotics
Programming
Electronics
Trouble Shooting
Debugging

At this point, the VisionArm is still being built, I continue working on the project and am still developing additional features for it to become a better project.

## Current Features for VisionArm

* Robotic Arm 5-DOF
* Robotic Arm actuated via Servo Motors
* Robotic Arm Controlled Using Xbox 360 Controller
* Robotic Arm Controlled Using Computer
* Robotic Arm Designed for Future Modifications and Expansions

## Future Developments

* Install a Vision System Using an Industrial Camera
